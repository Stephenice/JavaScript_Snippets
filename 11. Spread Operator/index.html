Topic: Spread Operator

Spread syntax can be used when all elements from an object or array need to be included in a list of some kind. used to build array and object and to pass multiple values into a function. we use the spread operator to expand an array into individual elements.

mostly used in the right side of assignment operator

iterables are: Array, String, Map, Set but not Objects. Most of the build in data structure are iterables.

when using spread operator in Array you are not manipulating the array(not mutating )

spread operator work in a place that execpt iterables values separated by comma.

array-like objects (objects with a length property and indexed elements); or iterable objects (objects such as Map and Set).

So the spread operator is used where we would otherwise write values, separated by a comma. On the other hand the rest pattern is basically used where we would otherwise write variable names separated by commas.
