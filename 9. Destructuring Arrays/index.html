Topic: Destructuring Arrays

Destructuring assignment syntax is a JavaScript expression that makes it possible to unpack values from arrays, or properties from objects, into distinct variables. When the values is unpack its become a variable.

Point to knows

  • When you Destructure an Array the (main) Array is not effected.
  • The values of the Destructure Array are called variable.
  • In an array destructuring from an array of length N specified on the right-hand side of the assignment, if the number of variables specified on the left-hand side of the assignment is greater than N, only the first N variables are assigned values. The values of the remaining variables will be undefined.
